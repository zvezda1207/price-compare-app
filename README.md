# 📊 Price Compare App

Приложение для автоматического сравнения прайсов поставщиков с эталонным файлом.

## 🚀 О проекте

Desktop-приложение на Python для автоматического сопоставления товаров и визуального сравнения цен нескольких поставщиков. Позволяет быстро определить наиболее выгодную цену без ручного анализа Excel-файлов.

Проект реализован как полноценное GUI-приложение и может быть собран в исполняемый файл (.exe).

## 🛠 Стек технологий

- Python 3
- pandas
- openpyxl
- tkinter
- PyInstaller

## 📋 Требования к входным файлам

Excel-файлы должны содержать следующие столбцы (названия указаны точно):

- **Наименование**
- **Цена за штуку**
- **Артикул**

## 📖 Инструкция по использованию

1. Запустите `PriceCompare.exe` (двойной клик) или `python gui.py`
2. Нажмите **«Выбрать эталон»** и выберите эталонный Excel-файл
3. Нажмите **«Выбрать поставщиков»** и выберите от 2 до 5 файлов поставщиков
4. Нажмите **«Выбрать место сохранения»** и укажите путь для итогового файла
5. Нажмите **«Сформировать файл»**
6. После завершения появится сообщение об успешном создании файла

> **Примечание:** при первом запуске возможна небольшая задержка (несколько секунд). Итоговый файл создаётся в формате .xlsx.

## ⚙ Функциональность

- Загрузка одного эталонного Excel-файла
- Загрузка от 2 до 5 файлов поставщиков
- **Сопоставление товаров:**
  - приоритет по артикулу
  - нормализация наименований (порядок слов не важен)
- Формирование итоговой таблицы
- Автоматическая подсветка минимальной цены
- Проверка структуры входных файлов
- Защита от некорректных данных
- Графический интерфейс
- Сборка в исполняемый файл (.exe)

## 📊 Результат

В итоговом Excel-файле:

- товары расположены в порядке эталонного прайса
- цены поставщиков отображаются в отдельных колонках
- минимальная цена автоматически выделяется цветом

## 🧠 Реализованная логика

- Нормализация строк через регулярные выражения
- Сопоставление через множества слов
- Валидация структуры входных данных
- Приведение цен к числовому формату
- Разделение логики на модули:
  - `matcher.py` — сопоставление
  - `utils.py` — нормализация
  - `excel_writer.py` — экспорт и форматирование
  - `gui.py` — пользовательский интерфейс

## 📦 Сборка .exe

```bash
pip install -r requirements.txt
pyinstaller --onefile --windowed gui.py
```

Исполняемый файл появится в папке `dist/`.

## 🎯 Преимущества

- Ускоряет выбор поставщика
