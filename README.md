# 📊 Price Compare App

Desktop-приложение для автоматического сравнения прайсов поставщиков с эталонным Excel-файлом.

## 🚀 О проекте

Приложение автоматизирует сопоставление товаров из нескольких Excel-файлов и формирует итоговую таблицу с визуальным выделением наиболее выгодной цены.

Решает типичную бизнес-задачу: сокращает время анализа прайсов и минимизирует риск ручных ошибок.

---

## 🛠 Стек технологий

- Python 3
- pandas
- openpyxl
- tkinter
- PyInstaller

---

## ✨ Ключевые возможности

- Загрузка 1 эталонного файла и 2–5 файлов поставщиков
- Сопоставление товаров:
  - приоритет по артикулу
  - нормализация наименований (порядок слов не важен)
- Формирование итоговой таблицы
- Автоматическая подсветка минимальной цены
- Валидация структуры входных файлов
- Обработка некорректных данных
- Графический интерфейс
- Сборка в standalone `.exe`

---

## 🧠 Архитектура и реализованная логика

- Нормализация строк с использованием регулярных выражений
- Сопоставление через множества слов
- Приоритетный поиск по артикулу
- Приведение цен к числовому формату
- Разделение логики на модули:


matcher.py — сопоставление товаров
utils.py — нормализация строк
excel_writer.py — экспорт и форматирование
gui.py — пользовательский интерфейс
main.py — точка входа
