  Price Compare — Сравнение прайсов поставщиков
═══════════════════════════════════════════════════════════

НАЗНАЧЕНИЕ
  Автоматическое сравнение цен нескольких поставщиков
  с эталонным прайсом. Позволяет быстро найти лучшую цену.

ТРЕБОВАНИЯ К ФАЙЛАМ
  Excel-файлы должны содержать столбцы (точно в таком виде):
    • Наименование
    • Цена за штуку
    • Артикул

ИНСТРУКЦИЯ
  1. Запустите PriceCompare.exe
  2. «Выбрать эталон» — выберите эталонный Excel
  3. «Выбрать поставщиков» — выберите 2–5 файлов поставщиков
  4. «Выбрать место сохранения» — укажите путь для результата
  5. «Сформировать файл» — дождитесь сообщения об успехе

РЕЗУЛЬТАТ
  • Товары в порядке эталона
  • Цены поставщиков в отдельных колонках
  • Минимальная цена выделена цветом

Примечание: при первом запуске возможна задержка 2–3 секунды.
